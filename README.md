### Installation
```
git clone git@github.com:po1inakoroleva/frontend-project-12.git
```
```
make install
```
## Running the app locally

```bash
make start
```

The app will be available on _localhost:3000_

## Deploy

Build the app

```bash
npm run build
```

Run builded app

```bash
npm run start
```
